## h3PointDistM[вҖӢ](#h3pointdistm "Direct link to h3PointDistM")

Returns the "great circle" or "haversine" distance between pairs of GeoCoord points (latitude/longitude) pairs in meters.

**Syntax**

```
h3PointDistM(lat1, lon1, lat2, lon2)

```

**Arguments**

- `lat1`, `lon1` вҖ” Latitude and Longitude of point1 in degrees. [Float64](/docs/sql-reference/data-types/float).
- `lat2`, `lon2` вҖ” Latitude and Longitude of point2 in degrees. [Float64](/docs/sql-reference/data-types/float).

**Returned values**

- Haversine or great circle distance in meters.[Float64](/docs/sql-reference/data-types/float).

**Example**

```
SELECT h3PointDistM(-10.0 ,0.0, 10.0, 0.0) AS h3PointDistM;

```
