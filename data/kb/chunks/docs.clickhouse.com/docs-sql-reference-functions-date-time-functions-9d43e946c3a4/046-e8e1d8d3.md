## toStartOfMicrosecond[вҖӢ](#toStartOfMicrosecond "Direct link to toStartOfMicrosecond")

Introduced in: v22\.6\.0

Rounds down a date with time to the start of the microseconds.

**Syntax**

```
toStartOfMicrosecond(datetime[, timezone])

```

**Arguments**

- `datetime` вҖ” Date and time. [`DateTime64`](/docs/sql-reference/data-types/datetime64)
- `timezone` вҖ” Optional. Timezone for the returned value. If not specified, the function uses the timezone of the `value` parameter. [`String`](/docs/sql-reference/data-types/string)

**Returned value**

Input value with sub\-microseconds [`DateTime64`](/docs/sql-reference/data-types/datetime64)

**Examples**

**Query without timezone**

```
WITH toDateTime64('2020-01-01 10:20:30.999999999', 9) AS dt64
SELECT toStartOfMicrosecond(dt64);

```

```
в”Ңв”Җв”Җв”Җв”ҖtoStartOfMicrosecond(dt64)в”Җв”җ
в”Ӯ 2020-01-01 10:20:30.999999000 в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

```

**Query with timezone**

```
WITH toDateTime64('2020-01-01 10:20:30.999999999', 9) AS dt64
SELECT toStartOfMicrosecond(dt64, 'Asia/Istanbul');

```


## toStartOfMinute[вҖӢ](#toStartOfMinute "Direct link to toStartOfMinute")

Introduced in: v1\.1\.0

Rounds down a date with time to the start of the minute.

NoteThe return type can be configured by setting [`enable_extended_results_for_datetime_functions`](/docs/operations/settings/settings#enable_extended_results_for_datetime_functions).

**Syntax**

```
toStartOfMinute(datetime)

```

**Arguments**

- `datetime` вҖ” A date with time to round. [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)

**Returned value**

Returns the date with time rounded down to the start of the minute. [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)

**Examples**

**Round down to the start of the minute**

```
SELECT
    toStartOfMinute(toDateTime('2023-04-21 10:20:30')),
    toStartOfMinute(toDateTime64('2023-04-21 10:20:30.5300', 8))
FORMAT Vertical

```
