[Edit this page](https://github.com/ClickHouse/ClickHouse/tree/master/docs/en/sql-reference/functions/date-time-functions.md)# Functions for Working with Dates and Times

Most functions in this section accept an optional time zone argument, e.g. `Europe/Amsterdam`. In this case, the time zone is the specified one instead of the local (default) one.

**Example**

```
SELECT
    toDateTime('2016-06-15 23:00:00') AS time,
    toDate(time) AS date_local,
    toDate(time, 'Asia/Yekaterinburg') AS date_yekat,
    toString(time, 'US/Samoa') AS time_samoa

```

## UTCTimestamp[вҖӢ](#UTCTimestamp "Direct link to UTCTimestamp")

Introduced in: v22\.11\.0

Returns the current date and time at the moment of query analysis. The function is a constant expression.

This function gives the same result that `now('UTC')` would. It was added only for MySQL support. [`now`](#now) is the preferred usage.

**Syntax**

```
UTCTimestamp()

```

**Aliases**: `UTC_timestamp`

**Arguments**

- None.

**Returned value**

Returns the current date and time at the moment of query analysis. [`DateTime`](/docs/sql-reference/data-types/datetime)

**Examples**

**Get current UTC timestamp**

```
SELECT UTCTimestamp()

```

## YYYYMMDDToDate[вҖӢ](#YYYYMMDDToDate "Direct link to YYYYMMDDToDate")

Introduced in: v23\.9\.0

Converts a number containing the year, month and day number to a `Date`.
This function is the opposite of function [`toYYYYMMDD()`](/docs/sql-reference/functions/date-time-functions#toYYYYMMDD).
The output is undefined if the input does not encode a valid Date value.

**Syntax**

```
YYYYMMDDToDate(YYYYMMDD)

```

**Arguments**

- `YYYYMMDD` вҖ” Number containing the year, month and day. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)

**Returned value**

Returns a `Date` value from the provided arguments [`Date`](/docs/sql-reference/data-types/date)

**Examples**

**Example**

```
SELECT YYYYMMDDToDate(20230911);

```

## YYYYMMDDToDate32[вҖӢ](#YYYYMMDDToDate32 "Direct link to YYYYMMDDToDate32")

This function is the opposite of function [`toYYYYMMDD()`](/docs/sql-reference/functions/date-time-functions#toYYYYMMDD).
The output is undefined if the input does not encode a valid `Date32` value.

**Syntax**

```
YYYYMMDDToDate32(YYYYMMDD)

```

**Arguments**

- `YYYYMMDD` вҖ” Number containing the year, month and day. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)

**Returned value**

Returns a `Date32` value from the provided arguments [`Date32`](/docs/sql-reference/data-types/date32)
