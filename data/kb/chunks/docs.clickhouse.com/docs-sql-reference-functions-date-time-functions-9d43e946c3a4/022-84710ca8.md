## makeDateTime[​](#makeDateTime "Direct link to makeDateTime")

Introduced in: v22\.6\.0

Creates a `DateTime` from year, month, day, hour, minute, and second, with optional timezone.

**Syntax**

```
makeDateTime(year, month, day, hour, minute, second[, timezone])

```

**Arguments**

- `year` — Year number. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `month` — Month number (1\-12\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `day` — Day of the month (1\-31\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `hour` — Hour (0\-23\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `minute` — Minute (0\-59\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `second` — Second (0\-59\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `timezone` — Timezone name. [`String`](/docs/sql-reference/data-types/string)

**Returned value**

Returns a `DateTime` value constructed from the provided arguments [`DateTime`](/docs/sql-reference/data-types/datetime)

**Examples**

**DateTime from year, month, day, hour, minute, second**

```
SELECT makeDateTime(2023, 2, 28, 17, 12, 33) AS DateTime;

```

```
┌────────────DateTime─┐
│ 2023-02-28 17:12:33 │
└─────────────────────┘

```

## makeDateTime64[​](#makeDateTime64 "Direct link to makeDateTime64")

Introduced in: v22\.6\.0

Creates a `DateTime64` from year, month, day, hour, minute, second, with optional fraction, precision, and timezone.

**Syntax**

```
makeDateTime64(year, month, day, hour, minute, second[, fraction[, precision[, timezone]]])

```

**Arguments**

- `year` — Year number. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `month` — Month number (1\-12\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `day` — Day of the month (1\-31\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `hour` — Hour (0\-23\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `minute` — Minute (0\-59\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `second` — Second (0\-59\). [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `fraction` — Fractional part of the second. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `precision` — Precision for the fractional part (0\-9\). [`UInt8`](/docs/sql-reference/data-types/int-uint)
- `timezone` — Timezone name. [`String`](/docs/sql-reference/data-types/string)

**Returned value**

Returns a `DateTime64` value constructed from the provided arguments [`DateTime64`](/docs/sql-reference/data-types/datetime64)

**Examples**

**DateTime64 from year, month, day, hour, minute, second**

```
SELECT makeDateTime64(2023, 5, 15, 10, 30, 45, 779, 5);

```

```
┌─makeDateTime64(2023, 5, 15, 10, 30, 45, 779, 5)─┐
│                       2023-05-15 10:30:45.00779 │
└─────────────────────────────────────────────────┘

```
