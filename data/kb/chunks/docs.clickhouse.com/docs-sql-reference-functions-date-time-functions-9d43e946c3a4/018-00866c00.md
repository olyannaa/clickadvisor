## fromDaysSinceYearZero[вҖӢ](#fromDaysSinceYearZero "Direct link to fromDaysSinceYearZero")

Introduced in: v23\.11\.0

For a given number of days elapsed since [1 January 0000](https://en.wikipedia.org/wiki/Year_zero), returns the corresponding date in the [proleptic Gregorian calendar defined by ISO 8601](https://en.wikipedia.org/wiki/Gregorian_calendar#Proleptic_Gregorian_calendar).

The calculation is the same as in MySQL's `FROM_DAYS()` function. The result is undefined if it cannot be represented within the bounds of the [Date](/docs/sql-reference/data-types/date) type.

**Syntax**

```
fromDaysSinceYearZero(days)

```

**Aliases**: `FROM_DAYS`

**Arguments**

- `days` вҖ” The number of days passed since year zero. [`UInt32`](/docs/sql-reference/data-types/int-uint)

**Returned value**

Returns the date corresponding to the number of days passed since year zero. [`Date`](/docs/sql-reference/data-types/date)

**Examples**

**Convert days since year zero to dates**

```
SELECT
fromDaysSinceYearZero(739136) AS date1,
fromDaysSinceYearZero(toDaysSinceYearZero(toDate('2023-09-08'))) AS date2

```
