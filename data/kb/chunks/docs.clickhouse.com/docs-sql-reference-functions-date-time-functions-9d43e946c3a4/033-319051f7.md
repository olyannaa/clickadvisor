## timezoneOf[​](#timezoneOf "Direct link to timezoneOf")

Introduced in: v21\.4\.0

Returns the timezone name of a [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64) value.

**Syntax**

```
timezoneOf(datetime)

```

**Aliases**: `timeZoneOf`

**Arguments**

- `datetime` — A value of type. [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)
- `timezone` — Optional. Timezone name to convert the `datetime` value's timezone to. [`String`](/docs/sql-reference/data-types/string)

**Returned value**

Returns the timezone name for `datetime` [`String`](/docs/sql-reference/data-types/string)

**Examples**

**Usage example**

```
SELECT timezoneOf(now());

```

```
┌─timezoneOf(now())─┐
│ Europe/Amsterdam  │
└───────────────────┘

```

## timezoneOffset[​](#timezoneOffset "Direct link to timezoneOffset")

Introduced in: v21\.6\.0

Returns the timezone offset in seconds from [UTC](https://en.wikipedia.org/wiki/Coordinated_Universal_Time).
The function takes daylight saving time and historical timezone changes at the specified date and time into account.

**Syntax**

```
timezoneOffset(datetime)

```

**Aliases**: `timeZoneOffset`

**Arguments**

- `datetime` — `DateTime` value to get the timezone offset for. [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)

**Returned value**

Returns the offset from UTC in seconds [`Int32`](/docs/sql-reference/data-types/int-uint)

**Examples**

**Usage example**

```
SELECT toDateTime('2021-04-21 10:20:30', 'America/New_York') AS Time,
toTypeName(Time) AS Type,
timezoneOffset(Time) AS Offset_in_seconds,
(Offset_in_seconds / 3600) AS Offset_in_hours;

```

```
┌────────────────Time─┬─Type─────────────────────────┬─Offset_in_seconds─┬─Offset_in_hours─┐
│ 2021-04-21 10:20:30 │ DateTime('America/New_York') │            -14400 │              -4 │
└─────────────────────┴──────────────────────────────┴───────────────────┴─────────────────┘

```

## toDayOfMonth[​](#toDayOfMonth "Direct link to toDayOfMonth")

Introduced in: v1\.1\.0

Returns the day of the month (1\-31\) of a `Date` or `DateTime`.

**Syntax**

```
toDayOfMonth(datetime)

```

**Aliases**: `DAY`, `DAYOFMONTH`

**Arguments**

- `datetime` — Date or date with time to get the day of month from. [`Date`](/docs/sql-reference/data-types/date) or [`Date32`](/docs/sql-reference/data-types/date32) or [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)

**Returned value**

Returns the day of the month of the given date/time [`UInt8`](/docs/sql-reference/data-types/int-uint)

**Examples**

**Usage example**

```
SELECT toDayOfMonth(toDateTime('2023-04-21 10:20:30'))

```

```
┌─toDayOfMonth(toDateTime('2023-04-21 10:20:30'))─┐
│                                              21 │
└─────────────────────────────────────────────────┘

```

## toDayOfWeek[​](#toDayOfWeek "Direct link to toDayOfWeek")

Introduced in: v1\.1\.0

Returns the number of the day within the week of a `Date` or `DateTime` value.

The two\-argument form of `toDayOfWeek()` enables you to specify whether the week starts on Monday or Sunday,
and whether the return value should be in the range from 0 to 6 or 1 to 7\.
