Converts a `DateTime` or `DateTime64` to the specified time zone.
The internal value (number of unix seconds) of the data doesn't change.
Only the value's time zone attribute and the value's string representation changes.

**Syntax**

```
toTimezone(datetime, timezone)

```

**Aliases**: `toTimeZone`

**Arguments**

- `date` — The value to convert. [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)
- `timezone` — The target time zone name. [`String`](/docs/sql-reference/data-types/string)

**Returned value**

Returns the same timestamp as the input, but with the specified time zone [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)

**Examples**

**Usage example**

```
SELECT toDateTime('2019-01-01 00:00:00', 'UTC') AS time_utc,
toTypeName(time_utc) AS type_utc,
toInt32(time_utc) AS int32utc,
toTimezone(time_utc, 'Asia/Yekaterinburg') AS time_yekat,
toTypeName(time_yekat) AS type_yekat,
toInt32(time_yekat) AS int32yekat,
toTimezone(time_utc, 'US/Samoa') AS time_samoa,
toTypeName(time_samoa) AS type_samoa,
toInt32(time_samoa) AS int32samoa
FORMAT Vertical;

```

## toUTCTimestamp[​](#toUTCTimestamp "Direct link to toUTCTimestamp")

Introduced in: v23\.8\.0

Converts a date or date with time value from one time zone to UTC timezone timestamp. This function is mainly included for compatibility with Apache Spark and similar frameworks.

**Syntax**

```
toUTCTimestamp(datetime, time_zone)

```

**Aliases**: `to_utc_timestamp`

**Arguments**

- `datetime` — A date or date with time type const value or an expression. [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)
- `time_zone` — A String type const value or an expression representing the time zone. [`String`](/docs/sql-reference/data-types/string)

**Returned value**

Returns a date or date with time in UTC timezone. [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64)

**Examples**

**Convert timezone to UTC**

```
SELECT toUTCTimestamp(toDateTime('2023-03-16'), 'Asia/Shanghai')

```

```
┌─toUTCTimestamp(toDateTime('2023-03-16'), 'Asia/Shanghai')─┐
│                                     2023-03-15 16:00:00 │
└─────────────────────────────────────────────────────────┘

```

## toUnixTimestamp[​](#toUnixTimestamp "Direct link to toUnixTimestamp")

Introduced in: v1\.1\.0

Converts a `String`, `Date`, or `DateTime` to a Unix timestamp (seconds since `1970-01-01 00:00:00 UTC`) as `UInt32`.

**Syntax**

```
toUnixTimestamp(date[, timezone])

```

**Arguments**

- `date` — Value to convert. [`Date`](/docs/sql-reference/data-types/date) or [`Date32`](/docs/sql-reference/data-types/date32) or [`DateTime`](/docs/sql-reference/data-types/datetime) or [`DateTime64`](/docs/sql-reference/data-types/datetime64) or [`String`](/docs/sql-reference/data-types/string)
- `timezone` — Optional. Timezone to use for conversion. If not specified, the server's timezone is used. [`String`](/docs/sql-reference/data-types/string)

**Returned value**

Returns the Unix timestamp. [`UInt32`](/docs/sql-reference/data-types/int-uint)
