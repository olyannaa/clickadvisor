## bitmapSubsetLimit[вҖӢ](#bitmapSubsetLimit "Direct link to bitmapSubsetLimit")

Introduced in: v20\.1\.0

Returns a subset of a bitmap from position `range_start` with at most `cardinality_limit` set bits. Uses 1\-based indexing.

**Syntax**

```
bitmapSubsetLimit(bitmap, range_start, cardinality_limit)

```

**Arguments**

- `bitmap` вҖ” Bitmap object. [`AggregateFunction(groupBitmap, T)`](/docs/sql-reference/data-types/aggregatefunction). \- `range_start` вҖ” Start of the range (inclusive). [`UInt32`](/docs/sql-reference/data-types/int-uint) \- `cardinality_limit` вҖ” Maximum cardinality of the subset. [`UInt32`](/docs/sql-reference/data-types/int-uint)

**Returned value**

Returns a bitmap containing at most `cardinality_limit` set bits, starting from `range_start` [`AggregateFunction(groupBitmap, T)`](/docs/sql-reference/data-types/aggregatefunction)

**Examples**

**Usage example**

```
SELECT bitmapToArray(bitmapSubsetLimit(bitmapBuild([1, 5, 3, 2, 8]), 3, 2)) AS res;

```

## bitmapToArray[вҖӢ](#bitmapToArray "Direct link to bitmapToArray")

Introduced in: v20\.1\.0

Converts a bitmap to an array of unsigned integers. It is the opposite of function [`bitmapBuild`](/docs/sql-reference/functions/bitmap-functions#bitmapBuild).

**Syntax**

```
bitmapToArray(bitmap)

```

**Arguments**

- `bitmap` вҖ” Bitmap to convert. [`AggregateFunction(groupBitmap, T)`](/docs/sql-reference/data-types/aggregatefunction).

**Returned value**

Returns an array of unsigned integers contained in the bitmap [`Array(UInt*)`](/docs/sql-reference/data-types/array)

**Examples**

**Usage example**

```
SELECT bitmapToArray(bitmapBuild([1, 2, 3, 4, 5])) AS res;

```

## bitmapTransform[вҖӢ](#bitmapTransform "Direct link to bitmapTransform")

Introduced in: v20\.1\.0

Changes up to N bits in a bitmap by swapping specific bit values in `from_array` with corresponding ones in `to_array`.

**Syntax**

```
bitmapTransform(bitmap, from_array, to_array)

```

**Arguments**

- `bitmap` вҖ” Bitmap object. [`AggregateFunction(groupBitmap, T)`](/docs/sql-reference/data-types/aggregatefunction). \- `from_array` вҖ” Array of original set bits to be replaced. [`Array(T)`](/docs/sql-reference/data-types/array). \- `to_array` вҖ” Array of new set bits to replace with. [`Array(T)`](/docs/sql-reference/data-types/array).
