│ 900150983CD24FB0D6963F7D28E17F72 │ └──────────────────────────────────┘ ``` ## RIPEMD160[​](#RIPEMD160 "Direct link to RIPEMD160") Introduced in: v24\.10\.0 Calculates the RIPEMD\-160 hash of the given string. **Syntax** ``` RIPEMD160(s) ``` **Arguments** - `s` — The input string to hash. [`String`](/docs/sql-reference/data-types/string) **Returned value**

Returns the RIPEMD160 hash of the given input string as a fixed\-length string. [`FixedString(20)`](/docs/sql-reference/data-types/fixedstring)

**Examples**

**Usage example**

```
SELECT HEX(RIPEMD160('The quick brown fox jumps over the lazy dog'));

```

```
┌─HEX(RIPEMD160('The quick brown fox jumps over the lazy dog'))─┐
│ 37F332F68DB77BD9D7EDD4969571AD671CF9DD3B                      │
└───────────────────────────────────────────────────────────────┘

```
