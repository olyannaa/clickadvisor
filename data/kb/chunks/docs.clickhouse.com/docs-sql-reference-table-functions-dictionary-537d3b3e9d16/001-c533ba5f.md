# dictionary \| ClickHouse Docs

- - [Functions](/docs/sql-reference/functions)- [Table functions](/docs/sql-reference/table-functions)- dictionary
[Edit this page](https://github.com/ClickHouse/ClickHouse/tree/master/docs/en/sql-reference/table-functions/dictionary.md)# dictionary

Displays the [dictionary](/docs/sql-reference/statements/create/dictionary) data as a ClickHouse table. Works the same way as [Dictionary](/docs/engines/table-engines/special/dictionary) engine.

## Syntax[​](#syntax "Direct link to Syntax")

```
dictionary('dict')

```

## Arguments[​](#arguments "Direct link to Arguments")

- `dict` — A dictionary name. [String](/docs/sql-reference/data-types/string).

## Returned value[​](#returned_value "Direct link to Returned value")

A ClickHouse table.

## Examples[​](#examples "Direct link to Examples")

Input table `dictionary_source_table`:

```
┌─id─┬─value─┐
│  0 │     0 │
│  1 │     1 │
└────┴───────┘

```

Create a dictionary:

```
CREATE DICTIONARY new_dictionary(id UInt64, value UInt64 DEFAULT 0) PRIMARY KEY id
SOURCE(CLICKHOUSE(HOST 'localhost' PORT tcpPort() USER 'default' TABLE 'dictionary_source_table')) LAYOUT(DIRECT());

```

```
SELECT * FROM dictionary('new_dictionary');

```

```
┌─id─┬─value─┐
│  0 │     0 │
│  1 │     1 │
└────┴───────┘

```
