## randNegativeBinomial[​](#randNegativeBinomial "Direct link to randNegativeBinomial")

Introduced in: v22\.10\.0

Returns a random Float64 number drawn from a [negative binomial distribution](https://en.wikipedia.org/wiki/Negative_binomial_distribution).

**Syntax**

```
randNegativeBinomial(experiments, probability[, x])

```

**Arguments**

- `experiments` — The number of experiments. [`UInt64`](/docs/sql-reference/data-types/int-uint)
- `probability` — `The probability of failure in each experiment as a value between` 0`and`1`. [`Float64\`](/sql\-reference/data\-types/float)
- `x` — Optional and ignored. The only purpose of the argument is to prevent [common subexpression elimination](/docs/sql-reference/functions/overview#common-subexpression-elimination) when the same function call is used multiple times in a query. [`Any`](/docs/sql-reference/data-types)

**Returned value**

Returns a random Float64 number drawn from the specified negative binomial distribution [`UInt64`](/docs/sql-reference/data-types/int-uint)

**Examples**

**Usage example**

```
SELECT randNegativeBinomial(100, .75) FROM numbers(5)

```

```
┌─randNegativeBinomial(100, 0.75)─┐
│                              33 │
│                              32 │
│                              39 │
│                              40 │
│                              50 │
└─────────────────────────────────┘

```

## randNormal[​](#randNormal "Direct link to randNormal")

Introduced in: v22\.10\.0

Returns a random Float64 number drawn from a [normal distribution](https://en.wikipedia.org/wiki/Normal_distribution).

**Syntax**

```
randNormal(mean, stddev[, x])

```

**Arguments**

- `mean` — The mean value of distribution [`Float64`](/docs/sql-reference/data-types/float)
- `stddev` — The standard deviation of the distribution [`Float64`](/docs/sql-reference/data-types/float)
- `x` — Optional and ignored. The only purpose of the argument is to prevent [common subexpression elimination](/docs/sql-reference/functions/overview#common-subexpression-elimination) when the same function call is used multiple times in a query. [`Any`](/docs/sql-reference/data-types)

**Returned value**

Returns a random Float64 number drawn from the specified normal distribution. [`Float64`](/docs/sql-reference/data-types/float)

**Examples**

**Usage example**

```
SELECT randNormal(10, 2) FROM numbers(5)

```

```
┌──randNormal(10, 2)─┐
│ 13.389228911709653 │
│  8.622949707401295 │
│ 10.801887062682981 │
│ 4.5220192605895315 │
│ 10.901239123982567 │
└────────────────────┘

```
