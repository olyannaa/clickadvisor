## randExponential[​](#randExponential "Direct link to randExponential")

Introduced in: v22\.10\.0

Returns a random Float64 number drawn from an [exponential distribution](https://en.wikipedia.org/wiki/Exponential_distribution).

**Syntax**

```
randExponential(lambda[, x])

```

**Arguments**

- `lambda` — Rate parameter or lambda value of the distribution [`Float64`](/docs/sql-reference/data-types/float)
- `x` — Optional and ignored. The only purpose of the argument is to prevent [common subexpression elimination](/docs/sql-reference/functions/overview#common-subexpression-elimination) when the same function call is used multiple times in a query. [`Any`](/docs/sql-reference/data-types)

**Returned value**

Returns a random Float64 number drawn from the specified exponential distribution. [`Float64`](/docs/sql-reference/data-types/float)

**Examples**

**Usage example**

```
SELECT randExponential(1/10) FROM numbers(5)

```

```
┌─randExponential(divide(1, 10))─┐
│              44.71628934340778 │
│              4.211013337903262 │
│             10.809402553207766 │
│              15.63959406553284 │
│             1.8148392319860158 │
└────────────────────────────────┘

```

## randFisherF[​](#randFisherF "Direct link to randFisherF")

Introduced in: v22\.10\.0

Returns a random Float64 number drawn from an [F\-distribution](https://en.wikipedia.org/wiki/F-distribution).

**Syntax**

```
randFisherF(d1, d2[, x])

```

**Arguments**

- `d1` — d1 degree of freedom in `X = (S1 / d1) / (S2 / d2)`. [`Float64`](/docs/sql-reference/data-types/float)
- `d2` — d2 degree of freedom in `X = (S1 / d1) / (S2 / d2)`. [`Float64`](/docs/sql-reference/data-types/float)
- `x` — Optional and ignored. The only purpose of the argument is to prevent [common subexpression elimination](/docs/sql-reference/functions/overview#common-subexpression-elimination) when the same function call is used multiple times in a query. [`Any`](/docs/sql-reference/data-types)

**Returned value**

Returns a random Float64 number drawn from the specified F\-distribution [`Float64`](/docs/sql-reference/data-types/float)

**Examples**

**Usage example**

```
SELECT randFisherF(10, 3) FROM numbers(5)

```

```
┌─randFisherF(10, 20)─┐
│  0.7204609609506184 │
│  0.9926258472572916 │
│  1.4010752726735863 │
│ 0.34928401507025556 │
│  1.8216216009473598 │
└─────────────────────┘

```

## randLogNormal[​](#randLogNormal "Direct link to randLogNormal")

Introduced in: v22\.10\.0

Returns a random Float64 number drawn from a [log\-normal distribution](https://en.wikipedia.org/wiki/Log-normal_distribution).

**Syntax**

```
randLogNormal(mean, stddev[, x])
```

**Arguments**

- `mean` — The mean value of distribution. [`Float64`](/docs/sql-reference/data-types/float)
- `stddev` — The standard deviation of the distribution. [`Float64`](/docs/sql-reference/data-types/float)
