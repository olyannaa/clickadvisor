## s2RectContains[вҖӢ](#s2rectcontains "Direct link to s2RectContains")

Determines if a given rectangle contains a S2 point. In the S2 system, a rectangle is represented by a type of S2Region called a `S2LatLngRect` that represents a rectangle in latitude\-longitude space.

**Syntax**

```
s2RectContains(s2PointLow, s2PointHi, s2Point)

```

**Arguments**

- `s2PointLow` вҖ” Low S2 point index corresponding to the rectangle. [UInt64](/docs/sql-reference/data-types/int-uint).
- `s2PointHigh` вҖ” High S2 point index corresponding to the rectangle. [UInt64](/docs/sql-reference/data-types/int-uint).
- `s2Point` вҖ” Target S2 point index. [UInt64](/docs/sql-reference/data-types/int-uint).

**Returned value**

- `1` вҖ” If the rectangle contains the given S2 point.
- `0` вҖ” If the rectangle doesn't contain the given S2 point.

**Example**

```
SELECT s2RectContains(5179062030687166815, 5177056748191934217, 5177914411069187297) AS rectContains;

```

## s2RectUnion[вҖӢ](#s2rectunion "Direct link to s2RectUnion")

Returns the smallest rectangle containing the union of this rectangle and the given rectangle. In the S2 system, a rectangle is represented by a type of S2Region called a `S2LatLngRect` that represents a rectangle in latitude\-longitude space.

**Syntax**

```
s2RectUnion(s2Rect1PointLow, s2Rect1PointHi, s2Rect2PointLow, s2Rect2PointHi)

```

**Arguments**

- `s2Rect1PointLow`, `s2Rect1PointHi` вҖ” Low and High S2 point indexes corresponding to the first rectangle. [UInt64](/docs/sql-reference/data-types/int-uint).
- `s2Rect2PointLow`, `s2Rect2PointHi` вҖ” Low and High S2 point indexes corresponding to the second rectangle. [UInt64](/docs/sql-reference/data-types/int-uint).

**Returned values**

- `s2UnionRect2PointLow` вҖ” Low S2 cell id corresponding to the union rectangle. [UInt64](/docs/sql-reference/data-types/int-uint).
- `s2UnionRect2PointHi` вҖ” High S2 cell id corresponding to the union rectangle. [UInt64](/docs/sql-reference/data-types/int-uint).

**Example**

```
SELECT s2RectUnion(5178914411069187297, 5177056748191934217, 5179062030687166815, 5177056748191934217) AS rectUnion;
```
