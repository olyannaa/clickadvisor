Determines if a cap contains a S2 point. A cap represents a part of the sphere that has been cut off by a plane. It is defined by a point on a sphere and a radius in degrees.

**Syntax**

```
s2CapContains(center, degrees, point)

```

**Arguments**

- `center` вҖ” S2 point index corresponding to the cap. [UInt64](/docs/sql-reference/data-types/int-uint).
- `degrees` вҖ” Radius of the cap in degrees. [Float64](/docs/sql-reference/data-types/float).
- `point` вҖ” S2 point index. [UInt64](/docs/sql-reference/data-types/int-uint).

**Returned value**

- `1` вҖ” If the cap contains the S2 point index. [UInt8](/docs/sql-reference/data-types/int-uint).
- `0` вҖ” If the cap doesn't contain the S2 point index. [UInt8](/docs/sql-reference/data-types/int-uint).

**Example**

```
SELECT s2CapContains(1157339245694594829, 1.0, 1157347770437378819) AS capContains;

```

## s2CapUnion[вҖӢ](#s2capunion "Direct link to s2CapUnion")

Determines the smallest cap that contains the given two input caps. A cap represents a portion of the sphere that has been cut off by a plane. It is defined by a point on a sphere and a radius in degrees.

**Syntax**

```
s2CapUnion(center1, radius1, center2, radius2)

```

**Arguments**

- `center1`, `center2` вҖ” S2 point indexes corresponding to the two input caps. [UInt64](/docs/sql-reference/data-types/int-uint).
- `radius1`, `radius2` вҖ” Radius of the two input caps in degrees. [Float64](/docs/sql-reference/data-types/float).

**Returned values**

- `center` вҖ” S2 point index corresponding the center of the smallest cap containing the two input caps. [UInt64](/docs/sql-reference/data-types/int-uint).
- `radius` вҖ” Radius of the smallest cap containing the two input caps. [Float64](/docs/sql-reference/data-types/float).

**Example**

```
SELECT s2CapUnion(3814912406305146967, 1.0, 1157347770437378819, 1.0) AS capUnion;

```

## s2RectAdd[вҖӢ](#s2rectadd "Direct link to s2RectAdd")

Increases the size of the bounding rectangle to include the given S2 point. In the S2 system, a rectangle is represented by a type of S2Region called a `S2LatLngRect` that represents a rectangle in latitude\-longitude space.

**Syntax**

```
s2RectAdd(s2pointLow, s2pointHigh, s2Point)

```

**Arguments**

- `s2PointLow` вҖ” Low S2 point index corresponding to the rectangle. [UInt64](/docs/sql-reference/data-types/int-uint).
- `s2PointHigh` вҖ” High S2 point index corresponding to the rectangle. [UInt64](/docs/sql-reference/data-types/int-uint).
- `s2Point` вҖ” Target S2 point index that the bound rectangle should be grown to include. [UInt64](/docs/sql-reference/data-types/int-uint).
