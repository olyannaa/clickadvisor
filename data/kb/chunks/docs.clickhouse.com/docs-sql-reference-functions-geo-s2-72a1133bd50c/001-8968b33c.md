## S2Index[​](#s2index "Direct link to S2Index")

[S2](https://s2geometry.io/) is a geographical indexing system where all geographical data is represented on a sphere (similar to a globe).

In the S2 library points are represented as the S2 Index \- a specific number which encodes internally a point on the surface of a unit sphere, unlike traditional (latitude, longitude) pairs. To get the S2 point index for a given point specified in the format (latitude, longitude) use the [geoToS2](#geotos2) function. Also, you can use the [s2ToGeo](#s2togeo) function for getting geographical coordinates corresponding to the specified S2 point index.

## geoToS2[​](#geotos2 "Direct link to geoToS2")

Returns [S2](#s2index) point index corresponding to the provided coordinates `(longitude, latitude)`.

**Syntax**

```
geoToS2(lon, lat)

```

**Arguments**

- `lon` — Longitude. [Float64](/docs/sql-reference/data-types/float).
- `lat` — Latitude. [Float64](/docs/sql-reference/data-types/float).

**Returned values**

- S2 point index. [UInt64](/docs/sql-reference/data-types/int-uint).

**Example**

```
SELECT geoToS2(37.79506683, 55.71290588) AS s2Index;

```

```
┌─────────────s2Index─┐
│ 4704772434919038107 │
└─────────────────────┘

```

## s2ToGeo[​](#s2togeo "Direct link to s2ToGeo")

Returns geo coordinates `(longitude, latitude)` corresponding to the provided [S2](#s2index) point index.

**Syntax**

```
s2ToGeo(s2index)

```

**Arguments**

- `s2index` — S2 Index. [UInt64](/docs/sql-reference/data-types/int-uint).

**Returned values**

- A [tuple](/docs/sql-reference/data-types/tuple) consisting of two values:
	- `lon`. [Float64](/docs/sql-reference/data-types/float).
	- `lat`. [Float64](/docs/sql-reference/data-types/float).

**Example**

```
SELECT s2ToGeo(4704772434919038107) AS s2Coodrinates;

```

```
┌─s2Coodrinates────────────────────────┐
│ (37.79506681471008,55.7129059052841) │
└──────────────────────────────────────┘

```

## s2GetNeighbors[​](#s2getneighbors "Direct link to s2GetNeighbors")

Returns S2 neighbor indexes corresponding to the provided [S2](#s2index). Each cell in the S2 system is a quadrilateral bounded by four geodesics. So, each cell has 4 neighbors.

**Syntax**

```
s2GetNeighbors(s2index)

```

**Arguments**

- `s2index` — S2 Index. [UInt64](/docs/sql-reference/data-types/int-uint).

**Returned value**

- An array consisting of 4 neighbor indexes: `array[s2index1, s2index3, s2index2, s2index4]`. [Array](/docs/sql-reference/data-types/array)([UInt64](/docs/sql-reference/data-types/int-uint)).

**Example**

```
SELECT s2GetNeighbors(5074766849661468672) AS s2Neighbors;

```

```
┌─s2Neighbors───────────────────────────────────────────────────────────────────────┐
│ [5074766987100422144,5074766712222515200,5074767536856236032,5074767261978329088] │
└───────────────────────────────────────────────────────────────────────────────────┘

```

## s2CellsIntersect[​](#s2cellsintersect "Direct link to s2CellsIntersect")

Determines if the two provided [S2](#s2index) cells intersect or not.

**Syntax**

```
s2CellsIntersect(s2index1, s2index2)

```

**Arguments**

- `siIndex1`, `s2index2` — S2 Index. [UInt64](/docs/sql-reference/data-types/int-uint).

**Returned value**

- `1` — If the cells intersect. [UInt8](/docs/sql-reference/data-types/int-uint).
- `0` — If the cells don't intersect. [UInt8](/docs/sql-reference/data-types/int-uint).

**Example**

```
SELECT s2CellsIntersect(9926595209846587392, 9926594385212866560) AS intersect;

```

```
┌─intersect─┐
│         1 │
└───────────┘

```
