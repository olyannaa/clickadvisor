[Edit this page](https://github.com/ClickHouse/ClickHouse/tree/master/docs/en/sql-reference/statements/index.md)# ClickHouse SQL Statements

Users interact with ClickHouse using SQL statements. ClickHouse supports common SQL statements like [SELECT](/docs/sql-reference/statements/select) and [CREATE](/docs/sql-reference/statements/create), but it also provides specialized statements like [KILL](/docs/sql-reference/statements/kill) and [OPTIMIZE](/docs/sql-reference/statements/optimize).
