multiple Bitcoin and Ethereum holdings that traded above $100\. It might be nice to remove duplicate rows (since they only repeat what we already know). You can add `DISTINCT` to `INTERSECT` to eliminate duplicate rows from the result:

```
SELECT crypto_name FROM holdings
INTERSECT DISTINCT
SELECT crypto_name FROM crypto_prices
WHERE price > 100;

```

```
┌─crypto_name─┐
│ Bitcoin     │
│ Ethereum    │
└─────────────┘

```
