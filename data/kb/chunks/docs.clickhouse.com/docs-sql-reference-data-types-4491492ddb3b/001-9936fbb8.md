[Edit this page](https://github.com/ClickHouse/ClickHouse/tree/master/docs/en/sql-reference/data-types/index.md)# Data Types in ClickHouse

This section describes the data types supported by ClickHouse, for example [integers](/docs/sql-reference/data-types/int-uint), [floats](/docs/sql-reference/data-types/float) and [strings](/docs/sql-reference/data-types/string).

System table [system.data\_type\_families](/docs/operations/system-tables/data_type_families) provides an
overview of all available data types.
It also shows whether a data type is an alias to another data type and its name is case\-sensitive (e.g. `bool` vs. `BOOL`).
