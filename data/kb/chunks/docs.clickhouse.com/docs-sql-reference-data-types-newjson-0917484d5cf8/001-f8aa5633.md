The `JSON` type stores JavaScript Object Notation (JSON) documents in a single column.

NoteIn ClickHouse Open\-Source JSON data type is marked as production ready in version 25\.3\. It's not recommended to use this type in production in previous versions.

To declare a column of `JSON` type, you can use the following syntax:

```
<column_name> JSON
(
    max_dynamic_paths=N,
    max_dynamic_types=M,
    some.path TypeName,
    SKIP path.to.skip,
    SKIP REGEXP 'paths_regexp'
)

```
