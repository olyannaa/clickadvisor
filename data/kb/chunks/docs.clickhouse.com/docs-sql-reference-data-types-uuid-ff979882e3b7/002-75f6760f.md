## Generating UUIDs[вҖӢ](#generating-uuids "Direct link to Generating UUIDs")

ClickHouse provides the [generateUUIDv4](/docs/sql-reference/functions/uuid-functions) function to generate random UUID version 4 values.

## Usage Example[вҖӢ](#usage-example "Direct link to Usage Example")

**Example 1**

This example demonstrates the creation of a table with a UUID column and the insertion of a value into the table.

```
CREATE TABLE t_uuid (x UUID, y String) ENGINE=TinyLog

INSERT INTO t_uuid SELECT generateUUIDv4(), 'Example 1'

SELECT * FROM t_uuid

```

## Restrictions[вҖӢ](#restrictions "Direct link to Restrictions")

The UUID data type only supports functions which [String](/docs/sql-reference/data-types/string) data type also supports (for example, [min](/docs/sql-reference/aggregate-functions/reference/min), [max](/docs/sql-reference/aggregate-functions/reference/max), and [count](/docs/sql-reference/aggregate-functions/reference/count)).

The UUID data type is not supported by arithmetic operations (for example, [abs](/docs/sql-reference/functions/arithmetic-functions#abs)) or aggregate functions, such as [sum](/docs/sql-reference/aggregate-functions/reference/sum) and [avg](/docs/sql-reference/aggregate-functions/reference/avg).
