``` ## trunc[​](#trunc "Direct link to trunc") Introduced in: v1\.1\.0 Like [`floor`](#floor) but returns the rounded number with the largest absolute value less than or equal to that of `x`. **Syntax** ``` truncate(x[, N]) ``` **Aliases**: `truncate` **Arguments**

- `x` — The value to round. [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal*`](/docs/sql-reference/data-types/decimal) or [`(U)Int*`](/docs/sql-reference/data-types/int-uint)
- `N` — Optional. The number of decimal places to round to. Defaults to zero, which means rounding to an integer. [`(U)Int*`](/docs/sql-reference/data-types/int-uint)

**Returned value**

Returns a rounded number of the same type as `x`. [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal*`](/docs/sql-reference/data-types/decimal) or [`(U)Int*`](/docs/sql-reference/data-types/int-uint)

**Examples**

**Basic usage**

```
SELECT truncate(123.499, 1) AS res;

```

```
┌───res─┐
│ 123.4 │
└───────┘

```
