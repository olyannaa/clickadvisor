## ceil[​](#ceil "Direct link to ceil")

Introduced in: v1\.1\.0

Like [`floor`](#floor) but returns the smallest rounded number greater than or equal to `x`.
If rounding causes an overflow (for example, `ceiling(255, -1)`), the result is undefined.

**Syntax**

```
ceiling(x[, N])

```

**Aliases**: `ceiling`

**Arguments**

- `x` — The value to round. [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal*`](/docs/sql-reference/data-types/decimal) or [`(U)Int*`](/docs/sql-reference/data-types/int-uint)
- `N` — Optional. The number of decimal places to round to. Defaults to zero, which means rounding to an integer. Can be negative. [`(U)Int*`](/docs/sql-reference/data-types/int-uint)

**Returned value**

Returns a rounded number of the same type as `x`. [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal*`](/docs/sql-reference/data-types/decimal) or [`(U)Int*`](/docs/sql-reference/data-types/int-uint)

**Examples**

**Basic usage**

```
SELECT ceiling(123.45, 1) AS rounded

```

```
┌─rounded─┐
│   123.5 │
└─────────┘

```

**Negative precision**

```
SELECT ceiling(123.45, -1)

```

```
┌─ceiling(123.45, -1)─┐
│                 130 │
└─────────────────────┘

```

## floor[​](#floor "Direct link to floor")

Introduced in: v1\.1\.0

Returns the largest rounded number less than or equal to `x`, where the rounded number is a multiple of `1 / 10 * N`, or the nearest number of the appropriate data type if `1 / 10 * N` isn't exact.

Integer arguments may be rounded with a negative `N` argument.
With non\-negative `N` the function returns `x`.

If rounding causes an overflow (for example, `floor(-128, -1)`), the result is undefined.

**Syntax**

```
floor(x[, N])

```

**Arguments**

- `x` — The value to round. [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal*`](/docs/sql-reference/data-types/decimal) or [`(U)Int*`](/docs/sql-reference/data-types/int-uint)
- `N` — Optional. The number of decimal places to round to. Defaults to zero, which means rounding to an integer. Can be negative. [`(U)Int*`](/docs/sql-reference/data-types/int-uint)

**Returned value**

Returns a rounded number of the same type as `x`. [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal*`](/docs/sql-reference/data-types/decimal) or [`(U)Int*`](/docs/sql-reference/data-types/int-uint)

**Examples**

**Usage example**

```
SELECT floor(123.45, 1) AS rounded

```

```
┌─rounded─┐
│   123.4 │
└─────────┘

```

**Negative precision**

```
SELECT floor(123.45, -1)

```

```
┌─floor(123.45, -1)─┐
│               120 │
└───────────────────┘

```
