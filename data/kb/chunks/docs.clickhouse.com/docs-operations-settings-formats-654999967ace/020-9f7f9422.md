## output\_format\_json\_pretty\_print[вҖӢ](#output_format_json_pretty_print "Direct link to output_format_json_pretty_print")

This setting determines how nested structures such as Tuples, Maps, and Arrays are displayed within the `data` array when using the JSON output format.

For example, instead of output:

```
"data":
[
  {
    "tuple": {"a":1,"b":2,"c":3},
    "array": [1,2,3],
    "map": {"a":1,"b":2,"c":3}
  }
],

```

The output will be formatted as:

```
"data":
[
    {
        "tuple": {
            "a": 1,
            "b": 2,
            "c": 3
        },
        "array": [
            1,
            2,
            3
        ],
        "map": {
            "a": 1,
            "b": 2,
            "c": 3
        }
    }
],

```

Enabled by default.

## output\_format\_json\_quote\_64bit\_floats[вҖӢ](#output_format_json_quote_64bit_floats "Direct link to output_format_json_quote_64bit_floats")

Controls quoting of 64\-bit [floats](/docs/sql-reference/data-types/float) when they are output in JSON\* formats.

Disabled by default.

## output\_format\_json\_quote\_64bit\_integers[вҖӢ](#output_format_json_quote_64bit_integers "Direct link to output_format_json_quote_64bit_integers")

Controls quoting of 64\-bit or bigger [integers](/docs/sql-reference/data-types/int-uint) (like `UInt64` or `Int128`) when they are output in a [JSON](/docs/interfaces/formats/JSON) format.
Such integers are enclosed in quotes by default. This behavior is compatible with most JavaScript implementations.

Possible values:

- 0 вҖ” Integers are output without quotes.
- 1 вҖ” Integers are enclosed in quotes.

## output\_format\_json\_quote\_decimals[вҖӢ](#output_format_json_quote_decimals "Direct link to output_format_json_quote_decimals")

Controls quoting of decimals in JSON output formats.

Disabled by default.

## output\_format\_json\_quote\_denormals[вҖӢ](#output_format_json_quote_denormals "Direct link to output_format_json_quote_denormals")

Enables `+nan`, `-nan`, `+inf`, `-inf` outputs in [JSON](/docs/interfaces/formats/JSON) output format.

Possible values:

- 0 вҖ” Disabled.
- 1 вҖ” Enabled.
