## input\_format\_defaults\_for\_omitted\_fields[​](#input_format_defaults_for_omitted_fields "Direct link to input_format_defaults_for_omitted_fields")

When performing `INSERT` queries, replace omitted input column values with default values of the respective columns. This option applies to [JSONEachRow](/docs/interfaces/formats/JSONEachRow) (and other JSON formats), [CSV](/docs/interfaces/formats/CSV), [TabSeparated](/docs/interfaces/formats/TabSeparated), [TSKV](/docs/interfaces/formats/TSKV), [Parquet](/docs/interfaces/formats/Parquet), [Arrow](/docs/interfaces/formats/Arrow), [Avro](/docs/interfaces/formats/Avro), [ORC](/docs/interfaces/formats/ORC), [Native](/docs/interfaces/formats/Native) formats and formats with `WithNames`/`WithNamesAndTypes` suffixes.

NoteWhen this option is enabled, extended table metadata are sent from server to client. It consumes additional computing resources on the server and can reduce performance.

Possible values:

- 0 — Disabled.
- 1 — Enabled.
