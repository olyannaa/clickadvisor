## timeSeriesDerivToGrid[‚Äã](#timeSeriesDerivToGrid "Direct link to timeSeriesDerivToGrid")

Introduced in: v25\.6\.0

Aggregate function that takes time series data as pairs of timestamps and values and calculates [PromQL\-like derivative](https://prometheus.io/docs/prometheus/latest/querying/functions/#deriv) from this data on a regular time grid described by start timestamp, end timestamp and step. For each point on the grid the samples for calculating `deriv` are considered within the specified time window.

NoteThis function is experimental, enable it by setting `allow_experimental_ts_to_grid_aggregate_function=true`.

**Syntax**

```
timeSeriesDerivToGrid(start_timestamp, end_timestamp, grid_step, staleness)(timestamp, value)

```

**Parameters**

- `start_timestamp` ‚Äî Specifies start of the grid. \- `end_timestamp` ‚Äî Specifies end of the grid. \- `grid_step` ‚Äî Specifies step of the grid in seconds. \- `staleness` ‚Äî Specifies the maximum "staleness" in seconds of the considered samples. The staleness window is a left\-open and right\-closed interval.

**Arguments**

- `timestamp` ‚Äî Timestamp of the sample. Can be individual values or arrays. \- `value` ‚Äî Value of the time series corresponding to the timestamp. Can be individual values or arrays.

**Returned value**

`deriv` values on the specified grid as an `Array(Nullable(Float64))`. The returned array contains one value for each time grid point. The value is NULL if there are not enough samples within the window to calculate the derivative value for a particular grid point.

**Examples**

**Calculate derivative values on the grid \[90, 105, 120, 135, 150, 165, 180, 195, 210]**

```
WITH
    -- NOTE: the gap between 140 and 190 is to show how values are filled for ts = 150, 165, 180 according to window parameter
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values, -- array of values corresponding to timestamps above
    90 AS start_ts,       -- start of timestamp grid
    90 + 120 AS end_ts,   -- end of timestamp grid
    15 AS step_seconds,   -- step of timestamp grid
    45 AS window_seconds  -- "staleness" window
SELECT timeSeriesDerivToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)
FROM
(
    -- This subquery converts arrays of timestamps and values into rows of `timestamp`, `value`
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);

```
