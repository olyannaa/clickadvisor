REPLACE TABLE myOldTable

SELECT * FROM myOldTable
WHERE CounterID <12345;

```

### Syntax[​](#syntax "Direct link to Syntax")

```
{CREATE [OR REPLACE] | REPLACE} TABLE [db.]table_name

```

NoteAll syntax forms for the `CREATE` statement also work for this statement. Invoking `REPLACE` for a non\-existent table will cause an error.

### Examples:[​](#examples "Direct link to Examples:")

- Local- Cloud

Consider the following table:
```
CREATE DATABASE base 
ENGINE = Atomic;

CREATE OR REPLACE TABLE base.t1
(
    n UInt64,
    s String
)
ENGINE = MergeTree
ORDER BY n;

INSERT INTO base.t1 VALUES (1, 'test');

SELECT * FROM base.t1;

┌─n─┬─s────┐
│ 1 │ test │
└───┴──────┘

```
We can use the `REPLACE` statement to clear all the data:
```
CREATE OR REPLACE TABLE base.t1 
(
    n UInt64,
    s Nullable(String)
)
ENGINE = MergeTree
ORDER BY n;

INSERT INTO base.t1 VALUES (2, null);

SELECT * FROM base.t1;

┌─n─┬─s──┐
│ 2 │ \N │
└───┴────┘

```
Or we can use the `REPLACE` statement to change the table structure:
```
REPLACE TABLE base.t1 (n UInt64) 
ENGINE = MergeTree 
ORDER BY n;

INSERT INTO base.t1 VALUES (3);

SELECT * FROM base.t1;

┌─n─┐
│ 3 │
└───┘

```
