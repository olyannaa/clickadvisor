## stddevPopStable[​](#stddevPopStable "Direct link to stddevPopStable")

Introduced in: v1\.1\.0

The result is equal to the square root of [varPop](/docs/sql-reference/aggregate-functions/reference/varPop). Unlike [stddevPop](/docs/sql-reference/aggregate-functions/reference/stddevpop), this function uses a numerically stable algorithm. It works slower but provides a lower computational error.

**Syntax**

```
stddevPopStable(x)

```

**Arguments**

- `x` — Population of values to find the standard deviation of. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal*`](/docs/sql-reference/data-types/decimal)

**Returned value**

Returns the square root of the variance of `x`. [`Float64`](/docs/sql-reference/data-types/float)

**Examples**

**Basic usage**

```
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population Float64,
)
ENGINE = Log;

INSERT INTO test_data SELECT randUniform(5.5, 10) FROM numbers(1000000);

SELECT
    stddevPopStable(population) AS stddev
FROM test_data;

```

```
┌─────────────stddev─┐
│ 1.2999977786592576 │
└────────────────────┘

```
