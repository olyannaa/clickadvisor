v21\.11\.0 Returns the uptime of the current ZooKeeper session in seconds. **Syntax** ``` zookeeperSessionUptime() ``` **Arguments** - None. **Returned value** Returns the uptime of the current ZooKeeper session in seconds. [`UInt32`](/docs/sql-reference/data-types/int-uint) **Examples** **Usage example** ``` SELECT zookeeperSessionUptime(); ```
