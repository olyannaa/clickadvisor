Builds a bar chart.
Draws a band with width proportional to (x \- min) and equal to width characters when x \= max.
The band is drawn with accuracy to one eighth of a symbol.

**Syntax**

```
bar(x, min, max[, width])

```

**Arguments**

- `x` вҖ” Size to display. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `min` вҖ” The minimum value. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `max` вҖ” The maximum value. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal`](/docs/sql-reference/data-types/decimal)
- `width` вҖ” Optional. The width of the bar in characters. The default is `80`. [`const (U)Int*`](/docs/sql-reference/data-types/int-uint) or [`const Float*`](/docs/sql-reference/data-types/float) or [`const Decimal`](/docs/sql-reference/data-types/decimal)

**Returned value**

Returns a unicode\-art bar string. [`String`](/docs/sql-reference/data-types/string)

**Examples**

**Usage example**

```
SELECT
toHour(EventTime) AS h,
count() AS c,
bar(c, 0, 600000, 20) AS bar
FROM test.hits
GROUP BY h
ORDER BY h ASC

```
