## polygonsIntersectionCartesian[​](#polygonsintersectioncartesian "Direct link to polygonsIntersectionCartesian")

Calculates the intersection of polygons.

### Example[​](#example-23 "Direct link to Example")

```
SELECT wkt(polygonsIntersectionCartesian([[[(0., 0.), (0., 3.), (1., 2.9), (2., 2.6), (2.6, 2.), (2.9, 1.), (3., 0.), (0., 0.)]]], [[[(1., 1.), (1., 4.), (4., 4.), (4., 1.), (1., 1.)]]]))

```

```
MULTIPOLYGON(((1 2.9,2 2.6,2.6 2,2.9 1,1 1,1 2.9)))

```

### Input parameters[​](#input-parameters-16 "Direct link to Input parameters")

Polygons

### Returned value[​](#returned-value-23 "Direct link to Returned value")

MultiPolygon

## polygonsUnionCartesian[​](#polygonsunioncartesian "Direct link to polygonsUnionCartesian")

Calculates the union of polygons.

### Example[​](#example-26 "Direct link to Example")

```
SELECT wkt(polygonsUnionCartesian([[[(0., 0.), (0., 3.), (1., 2.9), (2., 2.6), (2.6, 2.), (2.9, 1), (3., 0.), (0., 0.)]]], [[[(1., 1.), (1., 4.), (4., 4.), (4., 1.), (1., 1.)]]]))

```

```
MULTIPOLYGON(((1 2.9,1 4,4 4,4 1,2.9 1,3 0,0 0,0 3,1 2.9)))

```

### Input parameters[​](#input-parameters-19 "Direct link to Input parameters")

Polygons

### Returned value[​](#returned-value-26 "Direct link to Returned value")

MultiPolygon
