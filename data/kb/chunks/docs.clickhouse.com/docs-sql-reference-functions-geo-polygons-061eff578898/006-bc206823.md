## polygonsWithinCartesian[​](#polygonswithincartesian "Direct link to polygonsWithinCartesian")

Returns true if the second polygon is within the first polygon.

### Example[​](#example-18 "Direct link to Example")

```
SELECT polygonsWithinCartesian([[[(2., 2.), (2., 3.), (3., 3.), (3., 2.)]]], [[[(1., 1.), (1., 4.), (4., 4.), (4., 1.), (1., 1.)]]])

```

```
1

```

### Input parameters[​](#input-parameters-11 "Direct link to Input parameters")

Two polygons

### Returned value[​](#returned-value-18 "Direct link to Returned value")

UInt8, 0 for false, 1 for true

## polygonsIntersectCartesian[​](#polygonsintersectcartesian "Direct link to polygonsIntersectCartesian")

Returns true if the two polygons intersect (share any common area or boundary).

### Example[​](#example-intersects-cartesian "Direct link to Example")

```
SELECT polygonsIntersectCartesian([[[(2., 2.), (2., 3.), (3., 3.), (3., 2.)]]], [[[(1., 1.), (1., 4.), (4., 4.), (4., 1.), (1., 1.)]]])

```

```
1

```

### Input parameters[​](#input-parameters-intersects-cartesian "Direct link to Input parameters")

Two polygons

### Returned value[​](#returned-value-intersects-cartesian "Direct link to Returned value")

UInt8, 0 for false, 1 for true

## polygonsIntersectSpherical[​](#polygonsintersectspherical "Direct link to polygonsIntersectSpherical")

Returns true if the two polygons intersect (share any common area or boundary). Reference <https://www.boost.org/doc/libs/1_62_0/libs/geometry/doc/html/geometry/reference/algorithms/intersects.html>

### Example[​](#example-intersects-spherical "Direct link to Example")

```
SELECT polygonsIntersectSpherical([[[(4.3613577, 50.8651821), (4.349556, 50.8535879), (4.3602419, 50.8435626), (4.3830299, 50.8428851), (4.3904543, 50.8564867), (4.3613148, 50.8651279)]]], [[[(4.346693, 50.858306), (4.367945, 50.852455), (4.366227, 50.840809), (4.344961, 50.833264), (4.338074, 50.848677), (4.346693, 50.858306)]]]);

```

```
1

```

### Input parameters[​](#input-parameters-intersects-spherical "Direct link to Input parameters")

Two polygons

### Returned value[​](#returned-value-intersects-spherical "Direct link to Returned value")

UInt8, 0 for false, 1 for true

## polygonConvexHullCartesian[​](#polygonconvexhullcartesian "Direct link to polygonConvexHullCartesian")

Calculates a convex hull. [Reference](https://www.boost.org/doc/libs/1_61_0/libs/geometry/doc/html/geometry/reference/algorithms/convex_hull.html)

Coordinates are in Cartesian coordinate system.

### Example[​](#example-19 "Direct link to Example")

```
SELECT wkt(polygonConvexHullCartesian([[[(0., 0.), (0., 5.), (5., 5.), (5., 0.), (2., 3.)]]]))

```

```
POLYGON((0 0,0 5,5 5,5 0,0 0))

```
