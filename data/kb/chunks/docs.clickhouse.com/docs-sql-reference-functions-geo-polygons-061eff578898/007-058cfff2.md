## polygonAreaSpherical[​](#polygonareaspherical "Direct link to polygonAreaSpherical")

Calculates the surface area of a polygon.

### Example[​](#example-20 "Direct link to Example")

```
SELECT round(polygonAreaSpherical([[[(4.346693, 50.858306), (4.367945, 50.852455), (4.366227, 50.840809), (4.344961, 50.833264), (4.338074, 50.848677), (4.346693, 50.858306)]]]), 14)

```

```
9.387704e-8

```

### Input parameters[​](#input-parameters-13 "Direct link to Input parameters")

Polygon

### Returned value[​](#returned-value-20 "Direct link to Returned value")

Float
