Allows to perform queries on data stored in a [SQLite](/docs/engines/database-engines/sqlite) database.

## Syntax[​](#syntax "Direct link to Syntax")

```
sqlite('db_path', 'table_name')

```

## Arguments[​](#arguments "Direct link to Arguments")

- `db_path` — Path to a file with an SQLite database. [String](/docs/sql-reference/data-types/string).
- `table_name` — Name of a table in the SQLite database. [String](/docs/sql-reference/data-types/string).

## Returned value[​](#returned_value "Direct link to Returned value")

- A table object with the same columns as in the original `SQLite` table.

## Example[​](#example "Direct link to Example")

```
SELECT * FROM sqlite('sqlite.db', 'table1') ORDER BY col2;

```

```
┌─col1──┬─col2─┐
│ line1 │    1 │
│ line2 │    2 │
│ line3 │    3 │
└───────┴──────┘

```
