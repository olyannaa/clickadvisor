## varSamp[​](#varSamp "Direct link to varSamp")

Introduced in: v1\.1\.0

Calculate the sample variance of a data set.

The sample variance is calculated using the formula:

Σ(x−xˉ)2n−1\\frac{\\Sigma{(x \- \\bar{x})^2}}{n\-1}n−1Σ(x−xˉ)2​
  

Where:

- xxx is each individual data point in the data set
- xˉ\\bar{x}xˉ is the arithmetic mean of the data set
- nnn is the number of data points in the data set

The function assumes that the input data set represents a sample from a larger population. If you want to calculate the variance of the entire population (when you have the complete data set), you should use [`varPop`](/docs/sql-reference/aggregate-functions/reference/varPop) instead.

NoteThis function uses a numerically unstable algorithm. If you need [numerical stability](https://en.wikipedia.org/wiki/Numerical_stability) in calculations, use the [`varSampStable`](/docs/sql-reference/aggregate-functions/reference/varsampstable) function. It works slower but provides a lower computational error.

**Syntax**

```
varSamp(x)

```

**Aliases**: `VAR_SAMP`

**Arguments**

- `x` — The population for which you want to calculate the sample variance. [`(U)Int*`](/docs/sql-reference/data-types/int-uint) or [`Float*`](/docs/sql-reference/data-types/float) or [`Decimal*`](/docs/sql-reference/data-types/decimal)

**Returned value**

Returns the sample variance of the input data set `x`. [`Float64`](/docs/sql-reference/data-types/float)

**Examples**

**Computing sample variance**

```
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x Float64
)
ENGINE = Memory;

INSERT INTO test_data VALUES (10.5), (12.3), (9.8), (11.2), (10.7);

SELECT round(varSamp(x),3) AS var_samp FROM test_data;

```

```
┌─var_samp─┐
│    0.865 │
└──────────┘

```
