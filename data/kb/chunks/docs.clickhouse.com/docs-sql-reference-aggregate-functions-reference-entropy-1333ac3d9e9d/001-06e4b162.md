## entropy[​](#entropy "Direct link to entropy")

Introduced in: v20\.1\.0

Calculates the [Shannon entropy](https://en.wikipedia.org/wiki/Entropy_(information_theory)) for a column of values.

**Syntax**

```
entropy(val)

```

**Arguments**

- `val` — Column of values of any type. [`Any`](/docs/sql-reference/data-types)

**Returned value**

Returns Shannon entropy. [`Float64`](/docs/sql-reference/data-types/float)

**Examples**

**Basic entropy calculation**

```
CREATE TABLE entropy (`vals` UInt32,`strings` String)
ENGINE = Memory;

INSERT INTO entropy VALUES (1, 'A'), (1, 'A'), (1,'A'), (1,'A'), (2,'B'), (2,'B'), (2,'C'), (2,'D');

SELECT entropy(vals), entropy(strings) FROM entropy

```

```
└───────────────┴──────────────────┘

```
