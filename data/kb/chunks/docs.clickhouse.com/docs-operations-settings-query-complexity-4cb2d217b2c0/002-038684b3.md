Once reached, external sort is used.| [`max_rows_to_sort`](/docs/operations/settings/settings#max_rows_to_sort) The maximum number of rows before sorting. than `max_estimated_execution_time`.| [`max_execution_time_leaf`](/docs/operations/settings/settings#max_execution_time_leaf) Similar semantically to `max_execution_time` but only applied on leaf [`max_execution_speed`](/docs/operations/settings/settings#max_execution_speed) The maximum number of execution rows per second.|
