value. **Syntax** ``` toUnixTimestamp64Second(value) ``` **Arguments** - `value` — DateTime64 value with any precision. [`DateTime64`](/docs/sql-reference/data-types/datetime64) **Returned value** Returns a Unix timestamp in seconds. [`Int64`](/docs/sql-reference/data-types/int-uint) **Examples** **Usage example** ``` WITH toDateTime64('2025-02-13 23:31:31.011', 3, 'UTC') AS dt64 SELECT toUnixTimestamp64Second(dt64); ```
