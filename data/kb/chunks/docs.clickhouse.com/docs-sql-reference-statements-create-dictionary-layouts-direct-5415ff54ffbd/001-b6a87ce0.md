## direct[​](#direct "Direct link to direct")

The dictionary is not stored in memory and directly goes to the source during the processing of a request.

The dictionary key has the [UInt64](/docs/sql-reference/data-types/int-uint) type.

All types of [sources](/docs/sql-reference/statements/create/dictionary/sources#dictionary-sources), except local files, are supported.

Configuration example:

- DDL- Configuration file

```
LAYOUT(DIRECT())

```

```
<layout>
  <direct />
</layout>

```

## complex\_key\_direct[​](#complex_key_direct "Direct link to complex_key_direct")

This type of storage is for use with composite [keys](/docs/sql-reference/statements/create/dictionary/attributes#composite-key). Similar to `direct`.
