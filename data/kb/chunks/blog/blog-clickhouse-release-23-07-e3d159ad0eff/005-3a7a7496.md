discovery exercises \- Observability maybe? As well as being [well documented](https://prql-lang.org/book/), users can experiment with a [public playground](https://prql-lang.org/playground/). Let's consider a few simple examples using the UK house price dataset. Suppose wish to highest districts in in London

```
from uk_house_price

filter town == 'LONDON'
group district (              	
  aggregate {                    	
	avg_price = average price
  }
)
sort {-avg_price}
take 1..10


SELECT
	district,
	AVG(price) AS avg_price
FROM uk_house_price
WHERE town = 'LONDON'
GROUP BY district
ORDER BY avg_price DESC
LIMIT 10
```
