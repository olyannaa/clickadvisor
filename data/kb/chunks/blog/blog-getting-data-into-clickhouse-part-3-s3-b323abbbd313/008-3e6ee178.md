Traders are always mindful of a widening spread when trading a currency pair. It represents the highest potential cost and can lead to the [dreaded margin call](https://www.dailyfx.com/education/forex-trading-basics/what-is-margin-call-in-forex-trading.html) when combined with leverage. Sudden changes in the spread that can spell disaster are usually associated with world events causing volatility and a lack of liquidity in the market. Below, we look for the biggest day changes in the spread for the EUR/USD pair using a [window function](https://clickhouse.com/docs/en/sql-reference/window-functions/#clickhouse-specific-window-functions). In this case, we use the parquet files for example purposes. Note that this isnвҖҷt always as performant as using csv files since parquet files canвҖҷt be parsed in parallel, reducing our throughput slightly.

```

SELECT
    base,
    quote,
    day,
    spread - any(spread) OVER (PARTITION BY base, quote ORDER BY base ASC, quote ASC, day ASC ROWS BETWEEN 1 PRECEDING AND CURRENT ROW) AS change
FROM
(
    SELECT
        base,
        quote,
        avg(ask - bid) AS spread,
        day
    FROM s3Cluster('default', 'https://datasets-documentation.s3.eu-west-3.amazonaws.com/forex/parquet/year_month/*.parquet')
    WHERE (base = 'EUR') AND (quote = 'USD')
    GROUP BY
        base,
        quote,
        toYYYYMMDD(datetime) AS day
    ORDER BY
        base ASC,
        quote ASC,
        day ASC
)
ORDER BY change DESC
LIMIT 5
```
