The City of Westminster has been returned many times, which makes sense, as itвҖҷs a costly part of the city. LetвҖҷs say we only want to return each combination of `(town,district,type)` once. We could do this using the `LIMIT BY` syntax:

```

```
1SELECT town, district, type
2FROM uk.uk_price_paid
3WHERE county = 'GREATER LONDON' AND price > 10_000_000 AND type <> 'other'
4ORDER BY price DESC
5LIMIT 1 BY town, district, type
6LIMIT 10;
```

Alternatively, instead of having to list all field names after the `LIMIT BY`, we use `LIMIT BY ALL`:

```

```
1SELECT town, district, type
2FROM uk.uk_price_paid
3WHERE county = 'GREATER LONDON' AND price > 10_000_000 AND type <> 'other'
4ORDER BY price DESC
5LIMIT 1 BY ALL
6LIMIT 10;
```
