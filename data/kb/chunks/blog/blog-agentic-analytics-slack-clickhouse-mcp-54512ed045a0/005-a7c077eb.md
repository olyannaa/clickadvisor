\& testing [\#](/blog/agentic-analytics-slack-clickhouse-mcp#5-running--testing) We can run the example locally using `uv`: ``` uv run [main.py](http://main.py) INFO:slack_bolt.AsyncApp:A new session (s_xxxx) has been established INFO:slack_bolt.AsyncApp:⚡️ Bolt app is running! ``` In Slack, we can now ask our bot questions: ![mcp_slack_bot.gif](/uploads/mcp_slack_bot_81aa68071a.gif)

In your terminal, you should see output like the following. If you read through the output, you can understand how the LLM is interacting with MCP and ClickHouse.
