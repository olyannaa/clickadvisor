Featureform also offers mature (by design) integration points for offline stores, online stores, and vector databases. ## Getting started with Featureform and ClickHouse [\#](/blog/training-machine-learning-models-with-clickhouse-and-featureform#getting-started-with-featureform-and-clickhouse) First we install featureform with a simple pip installation: ``` pip install featureform ```

While Featureform employs a [modular architecture](https://docs.featureform.com/system-architecture) that can be [deployed in Kubernetes](https://docs.featureform.com/deployment/kubernetes), for our initial example, weвҖҷll use the excellent [getting started experience](https://docs.featureform.com/deployment/quickstart-docker). In this mode, all components of the architecture are deployed as a single docker container. Even better, a simple flag ensures that ClickHouse is deployed with a test dataset preloaded.

