# 44\.4 \- TTL Recompress example

## Example how to create a table and define recompression rules

```
CREATE TABLE hits
(
    `banner_id` UInt64,
    `event_time` DateTime CODEC(Delta, Default),
    `c_name` String,
    `c_cost` Float64
)
ENGINE = MergeTree
PARTITION BY toYYYYMM(event_time)
ORDER BY (banner_id, event_time)
TTL event_time + toIntervalMonth(1) RECOMPRESS CODEC(ZSTD(1)),
    event_time + toIntervalMonth(6) RECOMPRESS CODEC(ZSTD(6);

```
Default compression is LZ4\. See [the ClickHouseВ® documentation](https://clickhouse.com/docs/en/operations/server-configuration-parameters/settings#server-settings-compression)
for more information.

These TTL rules recompress data after 1 and 6 months.

CODEC(Delta, Default) вҖ“ **Default** means to use default compression (LZ4 \-\> ZSTD1 \-\> ZSTD6\) in this case.
