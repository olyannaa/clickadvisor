ClickHouse that doesn’t natively support window functions. Here’s an example demonstrating how to mimic a window function like runningDifference() using arrays: #### Step 1: Create Sample Data We’ll start by creating a test table with some sample data:

```
DROP TABLE IS EXISTS test_running_difference

CREATE TABLE test_running_difference
ENGINE = Log AS
SELECT 
    number % 20 AS id, 
    toDateTime('2010-01-01 00:00:00') + (intDiv(number, 20) * 15) AS ts, 
    (number * round(xxHash32(number % 20) / 1000000)) - round(rand() / 1000000) AS val
FROM numbers(100)
```
